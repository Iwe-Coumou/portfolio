# Portfolio Site & Static Site Generator

A static site generator built from scratch in Python that powers this portfolio. It parses Markdown files into an HTML node tree and generates a complete static site, deployed to GitHub Pages.

**GitHub:** [Iwe-Coumou/portfolio](https://github.com/Iwe-Coumou/portfolio)

**Live site:** [iwe-coumou.github.io/portfolio](https://iwe-coumou.github.io/portfolio/)

## What it does

- Parses Markdown into an internal node tree (inline and block elements)
- Converts the node tree to HTML using a custom recursive renderer
- Recursively copies static assets (images, CSS) to the output directory
- Generates pages from an HTML template with configurable base path
- Deployed automatically to GitHub Pages from the `docs` directory

## Implementation highlights

- Hand-rolled Markdown parser — no external libraries
- Recursive block and inline parsing (headings, lists, code, blockquotes, bold, italic, links, images)
- Configurable `basepath` for subdirectory deployment
- Recursive static file copy implemented manually without `shutil.copytree`
